# sedes-jornadas

## Lista de pasos semi ordenados para poner en funcionamiento

# npm install

# composer install

# copiar .env.example y pegarla en un nuevo archivo .env, adentro de /project

# mysql connection

# php artisan migrate:fresh

# php artisan key:generate

# php artisan db:seed

# npm run dev // para cargar frontend

# si se usa wamp, xammp o laragon, ir hacia la carpeta /public en la ruta del navegador
    Ejemplo: http://localhost/sedes-jornadas/project/public/
    sedes-jornadas en este ejemplo es una carpeta dentro de www
# si no se usa wamp, xammp o laragon, usar 
    php artisan serve
